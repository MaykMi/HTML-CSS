Tipos de links

Links externos

Você pode acessar o repositório publico no Github do guanabara.

É possivel acessar outras paginas web através da tag <a> e </a>. Onde o link (caminho) do site é inserido no atributo href="" e utilizando os atributos target="_blank" e rel="external" podemos manter a pagina atual aberta após abrir algum link para outra pagina.

Links internos

Esta é a primeira pagina do site, Se você quiser, pode acessar também a minha Segunda página.

É possível transitar entre paginas internas, tudo depende do caminho utilizado, porem para voltar normalmente é utilizado o comando ../ ou ./(tudo depende do o quanto quer volta) antes do caminho inserido no atributo href="" um exemplo de um tag de link para voltar a pagina anterior: <a> href="../link" ref="prev"</a>
