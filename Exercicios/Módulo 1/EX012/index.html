Formatos

É possivel utilizar o software handbreake para a conversão dos arquivos de vídeo

Videos hospedados localmente

Para videos se utiliza a tag <video> com o atributo poster para adicionar uma thumb, controls para dar o controle e é possivel diminuir a dimensão com o atributo width.
Dentro das tags de video, se utiliza a tag <source> com os atributos src="" e type="".

Videos hospedados externamente

videos do youtube

É possivel hospedar videos no youtube ou no vimeo e utilizar essas plataformas de muletas para não pesar o site, precisando apenas compartilhar e incorporar.
