Principais formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito utilizando a tag <b> e </b> (não semanântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> e </strong> (semântigo).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> e </i> (não semântica).

Nesta frase, temos um terpo em itálico usando a tag <em> e </em> (semântico).

Formatações novas pra mim

Texto marcado

Podemos criar também um texto marcado ultilizando a tag <mark> e </mark>.

É possivel mudar a marca sobre o texto utilizando o atributo "style="background-color: bisque;, onde a cor utilizada na marcação a cima ↑ é a "bisque"

E no outro parágrafo, temos outro texto marcado no final.

Porém, como vemos a cima ↑ o style de uma mark não se aplica a todas, é necessário abrir uma tag style no cabecário com um seletor para "mark".

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto grande utilizando a tag <big> e </big> (que se encontra obsoleta no momento).
E para textos pequenos se utiliza a tag <small> e </small>.

Texto deletado

Podemos marcar um texto como deletado para indicar que ele deve ser lido, mas não considerado.

É utilizada a tag <del> e </del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

É utilizada a tag <ins> e </ins>.

Também é possivel sublinhar o texto com a tag <u> e </u> (porém é não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3.

É utilizada a tag <sup> e </sup> para deixar algo sobrescrito. Como visto com o 20 no texto a cima ↑

Texto subescrito

Para inserir coisas do tipo H2O.

É utilizada a tag <sub> e </sub> para deixar algo subescrito. Como visto com o 2 no texto a cima ↑
